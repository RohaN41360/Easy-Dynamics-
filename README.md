# Easy Dynamics Extension

## Description

The Easy Dynamics Extension is a simple, user-friendly tool designed for debugging and managing records in a system that utilizes ribbon controls. This extension allows users to easily update records, manage lookup fields, and tweak legacy settings with an intuitive, modern interface.

## Features

- **Ribbon Debug**: Trigger the ribbon debugger functionality for testing and troubleshooting.
- **Update Record**: Update fields dynamically with options to add new fields.
- **Legacy Advance Settings**: Open the legacy settings for advanced configuration.
- **Update Lookup Fields**: Update fields related to lookups with a specific UI tailored for managing lookups.

## Requirements

- A modern web browser that supports Chrome Extensions (e.g., Google Chrome, Microsoft Edge).
- No server-side dependencies; the extension runs entirely in the browser.

## Installation

1. Clone this repository or download the ZIP file.
2. Open your browser and navigate to `chrome://extensions` (or equivalent in other browsers).
3. Enable **Developer Mode** (top-right corner).
4. Click on **Load unpacked**.
5. Select the folder where you downloaded or cloned the extension.
6. The extension will now be available in your browser!

## Usage

1. After installing the extension, click on the extension icon in the browser toolbar to open the UI.
2. Use the following buttons to perform different actions:
    - **Ribbon Debug**: Start the ribbon debugging process.
    - **Update Record**: Open a dialog to add and submit field updates.
    - **Open Legacy Advance Settings**: Open a modal for legacy settings.
    - **Update Lookup Fields**: Update lookup fields with the relevant information.
3. Each button has interactive animations and a smooth user experience.

## UI Features

- **Modern Design**: Clean, minimalistic design using modern colors and animations.
- **Dialogs**: Modals for updating records and managing lookup fields with easy-to-use input forms.
- **Responsiveness**: The UI is fully responsive and adapts to smaller screens, such as mobile devices.
